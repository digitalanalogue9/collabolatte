# Unified Prompt Pack — Marketing Website
Tone: calm, human, low‑pressure. No KPI or programme language. Use “opt‑in”, “no monitoring or evaluation”, “no recording or transcription”, “thoughtful matching”.
Image theme: apply Collabolatte UX/theme palette and mood. Use warm neutrals and sage/teal accents; avoid bright, high‑contrast, or neon palettes.
Palette references (from UX/theme):
- Backgrounds: #f6f5f2, #f0ede7, #e6e4df
- Text tones: #1f2428, #4f5b61
- Accents: #3a7f6c, #5d9a89, #6a9486
